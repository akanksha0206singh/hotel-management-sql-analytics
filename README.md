# Hotel Management SQL Analytics Project

## Overview

This project demonstrates the use of SQL for analyzing data from a hotel management system. The database simulates a real-world environment where information about hotels, rooms, guests, bookings, and employees is stored and analyzed to generate meaningful insights.

The goal of this project is to apply SQL techniques to perform business-oriented data analysis and demonstrate practical database design skills.

## Project Objectives

* Design a relational database schema for a hotel management system
* Perform analytical queries to extract business insights
* Analyze booking patterns, revenue trends, and guest behavior
* Demonstrate practical SQL concepts used in business analytics

## Project Structure

schema.sql – Contains SQL statements used to create database tables and define relationships.

data.sql – Includes sample data inserted into the database tables.

queries.sql – Contains analytical queries used to generate business insights from the dataset.

triggers.sql – Includes SQL triggers used to automate updates such as room status changes.

hotel_management_full_script.sql – full project script.

## Key Analysis Performed

* Revenue analysis by hotel and room type
* Monthly booking trends and occupancy patterns
* Guest segmentation and booking behavior
* Booking cancellation analysis
* Operational insights related to hotel management
